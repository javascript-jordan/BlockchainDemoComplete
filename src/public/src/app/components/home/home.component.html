Step 1 ( Add Users )

For step one we need to create users and add them to the blockchain. Give the user a name and age. Add as many users as you like. In the real world this would be the equivalent to registering on a stock or crypto currency exchange.


* Note: You cannot add users with the same name.

Step 2 ( Fund Wallets )

Now that we have our users made we need to fund wallets with cash. Add a currency and an amount for each user created earlier. In the real world this would be the equivalent to withdrawling funds from your bank and into the exchange you've registered with.


* Note: The max amount for a currency is 9999.

Step 3 ( Start Trading )

All of our users wallets have been funded now we can start executing trades with other users. A Real life scenario would be the equivilant of making a trade order on the exchange we just funded in step 2.


* Note: Users can not trade with themself and can only trade with currency they own.

Step 4 ( Inspect Blockchain )

Head on over to the Blockchain page to see how the transactions like adding a user, funding a wallet, and trading with other users gets recorded on the blockchain. Also take note of the block hashes and see how they correspond with one another. Their are public websites which enable us to see live transactions and blocks: Blockchain Explorer or Block Cypher.


* Note: If a transaction on the blockchain changes it will invalidate the state of the block and will be void.
